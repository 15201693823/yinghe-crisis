# 星贸纪元：荧河危机 - 游戏玩法说明

## 世界观概要

**联星历217年**。荧河空间站——坐落于三叉航道节点的自由港，巽风圈最重要的贸易枢纽。128年前建站，如今5200人在这里生活、争斗、求存。

三方势力撕扯了半个世纪：🏛️商会垄断了贸易渠道却喊着自由，⚒️矿工的血汗养活全站却活得最苦，🛡️总督在中间如履薄冰。航道说断就断，黑市在暗处膨胀，还有说不清道不明的"航道异常"——像是有什么东西在从另一端敲打。

你，星际联盟新派来的**谈判官**，踏入了这个漩涡。

## 游戏玩法

### 自由探索 · 故事驱动

这不是一个回合制游戏——你想走就走，想聊就聊，剧情随你的对话自然推进。

**操作方式：**
- **WASD / 方向键**：移动角色
- **E**：与NPC对话 / 进入传送门
- **Q**：打开故事日志
- **Esc**：关闭对话框

### NPC对话驱动剧情

每个NPC有独立性格、立场和秘密。你与他们的对话会：
1. **推进主线**——首次与关键NPC对话会触发剧情进展
2. **影响亲密度**——多聊天、做对NPC有利的事，亲密度上升
3. **解锁信息**——亲密度高了，NPC才会透露内部消息和秘密
4. **触发事件**——剧情推进到一定程度会触发航道断裂等关键事件

亲密度等级：
- **0-2**：冷漠，只说客套话
- **3-5**：正常，开始透露信息
- **6-8**：信任，分享内幕
- **9-10**：推心置腹，解锁隐藏内容

### 黑市解锁条件

暗市街区不是自动开放的。你需要同时满足以下任一条件：
1. ✔ 同时取得**陈老板**和**老赵**的信任（亲密度均>=3）
2. ✔ 或者与**阿杰**的亲密达到4级，他给你引荐

解锁后暗市传送门会在中央大厅出现，你可以在那里找到九姐和神秘客。

### 剧情事件

剧情随你的探索自然推进。关键事件包括：
- **航道断裂**——主线推进后触发，引发经济动荡
- **矿工罢工**——矿工满意度低时可能触发
- **暗线揭示**——故事后期探索暗市，可能触及荧河更深层的秘密

### 多结局系统

游戏根据你的选择和满意度走向不同结局：

| 结局 | 条件 | 说明 |
|------|------|------|
| 🌿 荧河平衡 | 三方满意度均>=60 | 三方共治，表面平静 |
| 🔮 光之河的渡口 | 触发暗线秘密 | 触及荧河的真正秘密 |
| 🛡️ 秩序的铁腕 | 支持总督最多，满意度>=70 | 集权维稳 |
| 🏛️ 市场之手 | 支持商会最多，满意度>=70 | 自由市场繁荣 |
| ⚒️ 矿工的尊严 | 支持矿工最多，满意度>=70 | 劳动者的胜利 |
| 🌑 腐烂的荧河 | 灰色选择过多或道德值<=10 | 暗市成了真正的主人 |
| 💥 危机的深渊 | 三方满意度均<30 | 全线崩溃，谈判失败 |

## AI角色

本游戏所有NPC对话由AI驱动（通过Coze工作流）。接入工作流后，NPC会：
- 根据当前剧情状态和亲密度调整语气和内容
- 自然融入荧河的经济和社会背景
- 对玩家输入做出合理反应

未接入Coze时自动使用本地回退对话，可流畅游玩。

## 给比赛评委的亮点

1. **AI驱动的角色扮演**：NPC有经济利益和立场，对话不是脚本
2. **自由探索无限制**：不设回合，玩家按自己的节奏推进
3. **多结局叙事**：6+种结局，玩家选择真正影响走向
4. **条件解锁机制**：黑市需要玩家主动经营人际关系才能解锁
